# VINFAST Backend

Backend APIs for VINFAST application built using FastAPI.

## Tech Stack
- Python (FastAPI)
- MySQL
- SQLAlchemy
- JWT Authentication

## Setup Instructions

```bash
git clone https://github.com/sandhyajaggu/vinfast-backend.git
cd vinfast-backend
python -m venv venv
venv\Scripts\activate
